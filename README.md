# SQL Database Practice

## Español

Proyecto diseñado para practicar conceptos fundamentales de **bases de datos relacionales** usando SQL.

### Contenido

- Creación de tablas
- Inserción de datos
- Consultas SQL
- Relaciones entre tablas

### Tecnologías

- SQL
- Bases de datos relacionales
- MySQL compatible

### Objetivo

Practicar consultas SQL y comprender la estructura de bases de datos relacionales.
